4    ## How to run
5    
6    Required launched PostgreSQL and installed Python3.10
7    
8    * copy config template
9    ```bash
10   cp -r config config
11   ```
12   * Fill config/config.yml in with required values 
13   * Create and activate venv
14   ```bash
15   python -m venv venv
16   source venv/bin/activate
17   ```
18   * install dependencies
19   ```bash
20   pip install -r requirements.txt
21   ```
22   * Fill in alembic.ini (probably only db url)
23   * apply migrations
24   ```bash
25   alembic upgrade head
26   ```
27   * ... and run
28   ```bash
29   python -m app
30   ```
